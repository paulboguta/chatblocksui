description: Components for AI chat interfaces. Built on shadcn/ui.
---

**chatblocks** gives you components that AI apps need - ai inputs, model selectors, streaming message, reasoning and more

Everything follows shadcn/ui patterns. If you know shadcn, you already know chatblocks.

Version 0.1 ships with chat essentials. Voice outputs, video outputs, agent builders and workflow components are coming next

[Produly opensource](https://github.com/paulboguta/chatblocksui)
